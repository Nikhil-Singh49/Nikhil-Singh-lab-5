Foodie - Food Delivery Website
Simple responsive food delivery site built with HTML & CSS.​

Features
Header with logo, nav & login/signup

Hero search form (location + category)

Food categories grid

Featured restaurants with ratings

Contact form
Mobile-responsive design

Quick Start
Save index.html and style.css

Add your images (food.jpg, pizzapalce.jpg, etc.)

Open index.html in browser
